Getting your team information in JSON form

1. Login to fantasy premier league via fantasy.premierleague.com using this browser
2. Navigate to the "Points" page
3. Copy the number in the URL after "fantasy.premierleague.com/entry/..."
4. Add this number to the following URL "https://fantasy.premierleague.com/api/my-team/YOUR NUMBER HERE/"
5. Copy and paste everything from the webpage, and paste it into the form on the MyFPL page!
